# Sales Analytics Dashboard - Power BI

## Project Overview
An interactive 4-page Sales Analytics Dashboard built in Power BI 
analyzing business performance across regions, categories and returns.

## Dashboard Pages
- *Page 1 - Executive Summary*: Total Sales 2.33M, Profit 292.30K, 
  Profit Margin 12.56%, Total Orders 5K
- *Page 2 - Regional Performance*: Sales breakdown by West, East, 
  Central and South regions
- *Page 3 - Category Analysis*: Technology, Furniture and 
  Office Supplies comparison
- *Page 4 - Returns Analysis*: Return rate 5.79%, 296 returned orders

## Tools & Skills Used
- Power BI Desktop
- DAX Functions
- Data Modeling
- Data Visualization

## Dataset
Superstore Sales Dataset
